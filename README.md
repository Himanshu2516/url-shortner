# 🔗 Short URL Generator

## 📌 Project Description

This is a URL Shortener web application built using **Node.js, Express, MongoDB, and EJS**.
It allows users to convert long URLs into short and easy-to-share links.

## 🚀 Features

* Generate short URLs from long links
* Redirect users to the original URL
* Store URLs in MongoDB database
* Simple and clean user interface

## 🛠 Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS

## 📂 Project Structure

controllers/

## ⚙️ Installation

1. Clone the repository
2. Install dependencies

```
npm install
```

3. Start the server

```
npm start
```
